## Run Locally

Clone the project

```bash
  git clone https://github.com/Lejoser40/react-native-todo-list.git
```

Go to the project directory

```bash
  cd react-native-todo-list
```

Install dependencies

```bash
  npm install
```

Start the server

```bash
  npm start
```
